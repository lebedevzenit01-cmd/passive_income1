# Калькулятор пассивного дохода

Готовый Vite/React-проект для публикации на Vercel.

## Проверенные настройки Vercel

- Framework Preset: Vite
- Install Command: npm install
- Build Command: npm run build
- Output Directory: dist

## Локальный запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```
